# Droste Effect Visualizer

A program to visualize the transformation of a standard Droste image into the mind-bending mathematical style of M.C. Escher's *Print Gallery*. 

## Core Visualizations
This tool breaks down the complex mathematics into two primary interactive modes:
* **Pixel Mapping:** Observe how every pixel are mapped into the Conformal Grid.
* **Mathematical Stages:** Showing how the original image is mapped into the Log Space, the rotation of the Log Space, and finally mapping back to the Cartesian plane to create the Escher effect.
